# Spotify-End-To-End-Data-Engineering-Project
Built a complete Data Engineering project from scratch using powerful tools like Azure Data Factory, Azure SQL DB, Azure Databricks, Unity Catalog, Delta Live Tables, Spark Streaming, PySpark, Databricks Asset Bundles, and GitHub. Applied real-world concepts like Dimensional Data Modeling and Slowly Changing Dimensions (SCDs).
